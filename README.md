# Netflix SQL Project

This project contains SQL practice queries written in SQL Server Management Studio (SSMS) using the Netflix titles dataset.

## Project overview
The purpose of this project is to practice basic and intermediate SQL skills on a real dataset.  
The queries focus on filtering, sorting, grouping, counting, pattern matching, and handling missing values.

## Tools used
- SQL Server Management Studio (SSMS)
- SQL Server
- GitHub

## Dataset
- Netflix titles dataset
- Main columns used: `title`, `type`, `director`, `cast`, `country`, `release_year`, `rating`

## SQL skills demonstrated
- SELECT
- WHERE
- ORDER BY
- GROUP BY
- COUNT
- LIKE
- NULL handling
- Filtering and sorting

## Files in this repository
- `README.md` - Project summary
- `netflix_queries.sql` - SQL practice queries
- `screenshots/` - Query and result screenshots

## Example queries
- Find all movies released after 2015
- Count how many TV Shows are in the table
- Show the top 10 most common ratings
- Find all titles where country contains India
- Find rows where director is missing
- Count titles released each year
- Show all titles sorted alphabetically
- Find titles where cast contains a specific actor
- Count how many titles belong to each type
